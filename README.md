# INvest - Stock Analysis Backtesting Engine

מערכת ניתוח מניות היסטורי שמבצעת "הליכה אחורה" בזמן - מנתחת מניות חודש אחר חודש
בהתבסס **אך ורק על נתונים שהיו זמינים באותה נקודת זמן**, ובודקת את ביצועי ההמלצות.

## מה המערכת עושה

1. **ניתוח טכני חודשי** - לכל חודש בשנה האחרונה, המערכת מחשבת אינדיקטורים טכניים (מומנטום, RSI, ממוצעים נעים, תנודתיות) עבור ~65 מניות מובילות
2. **דירוג וסינון** - מדרגת את המניות לפי ציון מרוכז ובוחרת את Top N
3. **מעקב ביצועים** - בודקת מה קרה בפועל חודש אחרי כל המלצה
4. **סטטיסטיקות** - אחוז הצלחה, תשואה ממוצעת, ביצוע לפי סקטור
5. **אירועים** - מציגה ציר זמן של אירועים גיאופוליטיים וכלכליים

## התקנה והרצה

```bash
# התקנת dependencies
pip install -r requirements.txt

# הרצת האפליקציה
python app.py
```

פתחו את הדפדפן ב-`http://localhost:5000`

## שיטת הניתוח

המערכת משתמשת באינדיקטורים טכניים כמותיים:
- **מומנטום** (1 ו-3 חודשים) - מגמת המחיר
- **RSI** (14 ימים) - האם המניה overbought/oversold
- **ממוצעים נעים** (SMA 20/50) - כיוון המגמה
- **תנודתיות** - רמת הסיכון
- **מגמת מחזור** - אישור למהלך המחיר

## הגבלות חשובות

- **אין הטיית ראייה לאחור** - כל ניתוח מבוסס רק על נתונים שהיו זמינים באותו זמן
- **אירועים לא צפויים** - מלחמות ומשברים לא ניתנים לחיזוי בניתוח טכני
- **ללא עלויות עסקה** - התוצאות לא כוללות עמלות ומיסים
- **תוצאות עבר אינן מבטיחות עתיד**
